
# nfportalutils

<!-- badges: start -->
<!-- badges: end -->

The goal of nfportalutils is to provide convenience functions for updating NF-OSI file metadata and portal data. 

## Installation

You can install nfportalutils from here:

``` r
remotes::install_github("nf-osi/nfportalutils")
```
